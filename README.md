# Instagram Following Back Checker

This is a *very* simple app that helps you determine who is **not following you back** on Instagram.

## Prerequisites

Before running the app, request your Instagram data export from:

`accountscenter.meta.com/info_and_permissions`

Use these settings:

- **Format:** JSON
- **Date range:** All time

Optional (recommended for a smaller export):

- Customize information to only include **Connections > Followers and Following**

Download the zip file and extract before using the app.

## Dependencies

This project requires Python 3. No additional libraries are needed for the core functionality.
